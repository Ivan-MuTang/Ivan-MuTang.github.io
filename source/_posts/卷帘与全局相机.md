### 一、工作原理

- **全局曝光（Global Shutter）**
    
    所有像素**同时开始、同时结束**曝光；曝光结束后再统一读出。
    
    ![](data:image/svg+xml,%3csvg%20xmlns=%27http://www.w3.org/2000/svg%27%20version=%271.1%27%20width=%27256%27%20height=%27192%27/%3e)<img src="https://tc.jisicn.top/img/ebbd68acceb3b4b53842324d8f7248e9~tplv-be4g95zd3a-image.jpeg" width="300" height="225">
    
    类比：全班一起喊 “茄子”，同步拍照。
    
- **卷帘曝光（Rolling Shutter）**
    
    像素**从上到下逐行**曝光 + 读出；行与行有微秒级时间差。
    
    ![](data:image/svg+xml,%3csvg%20xmlns=%27http://www.w3.org/2000/svg%27%20version=%271.1%27%20width=%27256%27%20height=%27192%27/%3e)<img src="https://tc.jisicn.top/img/49b35a228e66f31eb7463e1aca0639c9~tplv-be4g95zd3a-image.jpeg" width="512" height="384">
    
    类比：按排依次拍照，前后排有时间差。
    

---

### 二、核心差异（一目了然）

表格

| 对比项          | 全局曝光              | 卷帘曝光              |
| ------------ | ----------------- | ----------------- |
| **曝光时序**     | 所有像素同步            | 逐行异步，有时间差         |
| **运动成像**     | 无畸变、无果冻效应         | 高速运动易倾斜 / 弯曲 / 拖影 |
| **频闪 / 灯光**  | 稳定、无条纹            | 易出现滚动条纹、亮度不均      |
| **传感器结构**    | 复杂（4T/5T 像素 + 存储） | 简单（3T 像素），填充因子高   |
| **分辨率 / 帧率** | 同面积下像素数较低         | 易高分辨率、高帧率（60fps+） |
| **噪声 / 感光**  | 噪声略高、感光稍弱         | 噪声低、低光表现好         |
| **成本**       | 高                 | 低，量产普及            |

---

### 三、关键现象：果冻效应

- **卷帘曝光**：因逐行时间差，高速运动或相机抖动时，物体出现**倾斜、弯曲、拉伸**变形（如旋转的风扇变弧形、行驶的车倾斜）。
    
    ![](data:image/svg+xml,%3csvg%20xmlns=%27http://www.w3.org/2000/svg%27%20version=%271.1%27%20width=%27256%27%20height=%27192%27/%3e)<img src="https://tc.jisicn.top/img/a408817f2178406ce77b6a18ad5b8cbe~tplv-be4g95zd3a-image.jpeg" width="512" height="384">
    
- **全局曝光**：同一时刻成像，**完全无果冻效应**。

---

### 四、典型应用场景

- ✅ **全局曝光更适合**
    
    - 高速运动拍摄：流水线检测、飞拍、体育 / 赛车、无人机高速抓拍。
    - 频闪光源环境：工业 LED 频闪照明、荧光灯下成像。
    - 高稳定要求：震动设备、精密测量、二维码高速读取。
    
- ✅ **卷帘曝光更适合**
    
    - 静态 / 低速场景：日常拍照、视频通话、监控（低速）。
    - 高分辨率 / 低光：手机、消费级相机、安防摄像头。
    - 成本敏感：量产设备、家用 / 民用产品。
    

---

### 五、快速记忆

- **全局 = 同步 = 稳 = 贵**：运动无变形，频闪不闪烁，成本高。
- **卷帘 = 逐行 = 易变形 = 便宜**：高像素低光好，运动易果冻，成本低
